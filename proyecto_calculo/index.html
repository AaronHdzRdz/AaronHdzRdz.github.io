🎯 Objetivo

Determinar el número óptimo de suéteres que deben producirse y venderse diariamente para maximizar las ganancias del negocio familiar, utilizando herramientas de cálculo diferencial.

💡 Justificación

En el negocio familiar de Celia Uniformes, optimizar la producción puede aumentar las ganancias y reducir costos innecesarios. Con este proyecto, se aplicarán conceptos matemáticos para encontrar un equilibrio entre los costos de producción, las ventas y los ingresos, obteniendo un modelo matemático práctico.

📚 Marco Teórico

Funciones involucradas

  • Costo de producción (C(x)):

    Representa el costo total de producir x suéteres. Incluye costos fijos y variables.

    Ejemplo: $$C(x) = 150x + 500$$

    Donde:

    • 150x: costo variable (precio por suéter)
    • 500: costo fijo diario (electricidad, renta, etc.)
  • Ingreso por ventas (I(x)):

    Representa el dinero obtenido al vender x suéteres.

    Ejemplo: $$I(x) = 300x$$

    Donde 300 es el precio promedio de venta de cada suéter.

  • Ganancia total (G(x)):

    Representa la diferencia entre el ingreso y el costo.

    $$G(x) = I(x) - C(x)$$

Máximos y mínimos de una función

Para maximizar la ganancia, se utiliza la primera derivada de G(x):

  • La derivada (G'(x)) se iguala a 0 para encontrar puntos críticos.
  • Se verifica si es un máximo con la segunda derivada (G''(x)).

🔍 Metodología

  1. Identificar las funciones:
    • Determinar los costos fijos y variables del negocio.
    • Estimar el precio promedio de venta de un suéter.
  2. Definir la función de ganancia:

    Sustituir las funciones de costos e ingresos en la ecuación: $$G(x) = I(x) - C(x)$$

  3. Derivar la función:
    • Calcular la primera derivada G'(x): $$G'(x) = I'(x) - C'(x)$$
    • Resolver $$G'(x) = 0$$ para encontrar x (punto crítico).
  4. Verificar el máximo:

    Usar la segunda derivada G''(x). Si $$G''(x) < 0$$, el punto crítico es un máximo.

  5. Interpretar resultados:
    • Calcular el número óptimo de suéteres a producir y vender.
    • Determinar la ganancia máxima.

🧮 Ejemplo Numérico

Datos del negocio:

  • Costo fijo diario: 500 pesos
  • Costo por suéter: 150 pesos
  • Precio de venta por suéter: 300 pesos

Funciones:

  • Costo de producción: $$C(x) = 150x + 500$$
  • Ingreso por ventas: $$I(x) = 300x$$
  • Ganancia total: $$G(x) = I(x) - C(x) = 300x - (150x + 500)$$

Simplificando: $$G(x) = 150x - 500$$

Derivada de la ganancia:

Primera derivada: $$G'(x) = 150$$

Como G'(x) es constante y positiva, la ganancia aumenta linealmente. Para maximizar la producción, se deben considerar restricciones externas, como capacidad de producción o demanda.

Análisis avanzado con restricciones

Si se considera una restricción de capacidad máxima de 15 suéteres diarios, la ganancia óptima es:

$$G(15) = 150(15) - 500 = 2250 - 500 = 1750$$ pesos diarios

📊 Resultados Esperados

  • La cantidad óptima de suéteres a producir diariamente (según demanda y capacidad).
  • El máximo beneficio que puede obtenerse bajo las condiciones actuales.

🏁 Conclusiones

Este proyecto demuestra cómo las herramientas de cálculo pueden aplicarse para tomar decisiones en un negocio real. Optimizar la producción ayuda a maximizar las ganancias y mejorar la eficiencia del negocio.
